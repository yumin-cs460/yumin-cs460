Git Setup

After download and install git, first thing I initial repository and then start "add" and "commit" my stuffs. To able code remotely, I also push it to my github. Code below show my steps:

               
                 git init          
                 git add .
                 git commit -m "XXXX"
                 git remote add origin https://github.com/xxx/xxx.git
                 git push origin master

When I need to access and change my code remotely, I clone the repository to my local machine. Code below show the step:

               
                 git clone https://github.com/XXX/XXX.git

Bootstrap Setup

I choose to save the bootstrap to local, code below show how I apply it to my page:

               
                 <!-- Bootstrap core CSS -->
                 <link href="vendor/bootstrap/css/bootstrap.min.css" rel="stylesheet">
                 <!-- Bootstrap core JavaScript -->
                 <script src="vendor/jquery/jquery.min.js"></script>
                 <script src="vendor/bootstrap/js/bootstrap.bundle.min.js"></script>
                 <link rel="stylesheet" type="text/css" href="main.css">

Navigation Bar

Use <nav> and <button> element to create navigation bar and button for navigating to different page. Code show the steps:

               
                 <!-- Navigation -->
                 <nav class="navbar navbar-expand-lg navbar-dark bg-primary static-top">
                   <div class="container">
                     <a class="navbar-brand" href="index.html">Yumin Chen</a>
                     <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarResponsive" aria-controls="navbarResponsive" aria-expanded="false" aria-label="Toggle navigation">
                       <span class="navbar-toggler-icon"></span>
                       </button>
                          <div class="collapse navbar-collapse" id="navbarResponsive">
                            <ul class="navbar-nav ml-auto">
                            <li class="nav-item active">
                            <a class="nav-link" href="#">Home
                          <span class="sr-only">(current)</span>
                             </a>
                             </li>
                             <li class="nav-item">
                        <a class="nav-link" href="page2.html">Picture</a>
                       </li>
                     </ul>
                   </div>
                  </div>
                 </nav>

Custom CSS

To able to let both of pages to use external custom CSS file, those pages will need to make a declaration so the custom CSS can effect. Code show the steps:

               
                <link rel="stylesheet" type="text/css" href="layout.css">

And then the external CSS can start adust the pages:

               
                h1, h2, h3, h4, h5, h6 {
                     font-size: 150%;
                }

                ul, ol {
                     list-style: upper-roman;
                }
