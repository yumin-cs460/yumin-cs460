Git Setup

After download and install git, first thing I initial repository and addand commit my stuff. To able code remotely, I also push it to my github. Code below show my steps:

               
                 git init          
                 git add .
                 git commit -m "XXXX"
                 git remote add origin https://github.com/xxx/xxx.git
                 git push origin master

When I need to access and change my code remotely, I clone the repository to my local machine. Code below show the step:

               
                 git clone https://github.com/yumin-cs460/cs460hw1.git
