Git Branch

For this HW, all the codes is in different branch and will be merge when finish, follow code show how to create and shift to different branch:

               
                 git clone https://github.com/xxx/xxx.git
                 git branch hw2_js          
                 git checkout hw2_js
                 git add hw2
                 git commit -m "XXXX"
                 git push origin hw2_js

Aso user follow code to view your remote branch and checkout the branch:

               
                  git branch -a  
                  git checkout hw2_js

Custom CSS

To able to let both of pages to use external custom CSS file, those pages will need to make a declaration so the custom CSS can effect. Code show the steps:

               
                <link rel="stylesheet" type="text/css" href="layout.css">

And then the external CSS can start adust the pages:

               
                h1, h2, h3, h4, h5, h6 {
                     font-size: 150%;
                }

                ul, ol {
                     list-style: upper-roman;
                }
