Git Branch and .gitignore

First time first, create another git branch for this HW assignment and switch to the new branch. Code below show the steps:

               
                 git clone https://github.com/xxx/xxx.git
                 git branch hw3          
                 git checkout hw3
                 git add HW3
                 git commit -m "XXXX"
                 git push origin hw3

The .gitignore file specifies untracked Visual Studio files that Git should ignore. The VS gitignore templates is available on Github gitignore repositories.

XML Documentation Comments

Not like Java, C# use XML for comments and the comment field is indicated by triple slashes(///). Visual Studio will also create(Off by default) a XML documentation file that included all the comments. Follow show the XML example and generated XML file.

                /// Comments Test 123
